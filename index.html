歡迎來到鳳梨小宇宙

真實生活的酸、甜、刺，都是最精彩的日常。

Hello, 我是 Amy

我是一位熱愛內容創作與行銷專業的創作者。曾於香港與北京深耕多年，參與過多場大型活動策劃與演唱會執行，這些豐富的跨城市經驗，讓我對品牌經營與市場脈動有著敏銳的觀察力。

目前，我專注於經營 「AIMEE Sweet and Sour Routine」 個人品牌。透過「鳳梨」這個象徵，記錄生活中的酸甜與挑戰。對我而言，生活或許有時帶刺，但只要用心經營，內裡始終甘甜。

除了行銷與影音剪輯，我也正沉浸在 Java 程式設計的邏輯世界中。這裡記錄了我的跨領域轉職筆記、美食足跡以及健身日記。
